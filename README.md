# Klo.oor

This is an app for call controll in companies that require their employees to communicate their equipament needs.

## Figma

The prototyping is here: [https://www.figma.com/design/NixiWPabFvgoUjlctjMWjd/klo.oor?node-id=0-1&t=3g6tmKviBJyUAuBK-1](https://www.figma.com/design/NixiWPabFvgoUjlctjMWjd/klo.oor?node-id=0-1&t=3g6tmKviBJyUAuBK-1)

## Requirements

### Screens
- Login
- List of all calls (viewable by the manager)
- List of all calls (viewable by the employee)
- Employee registration
- Call registration
- Call details

### Roles
- Manager
- Employee

### Features for Manager
- Login as manager
- Register an employee
- See calls
- Close calls

### Features for Employee
- Login as employee
- Open a call
- See your calls

### Details of a Call
- Employee name
- Employee department
- Employee email
- Employee phone
- Equipment name
- Equipment number (optional)
- Problem description

## How to run

Start an Apache server for PHP and MySQL with XAMPP.

Go to the MySQL control panel, look for the SQL tab, and paste the schema.sql file into the bd folder, then run it.

Now all you have to do is run the server! The administrator password is:
- Email: admin@gmail.com
- Password: 1234
